# AI 검색 인사이트 리포트 — 환자 질의 패턴 × 인용 매체 분석

> 조사일: 2026-07-03 · 웹 리서치 30회(한국어·영어) 종합.
> 이 문서는 메디픽 콘텐츠·생성기 설계의 근거 자료이며, 분기마다 재조사해 갱신한다.

---

## 1. 환자는 AI에 이렇게 묻는다

### 시장 규모 (한국)
- 챗GPT 국내 MAU **약 2,162만 명** (2025.11, 스마트폰 사용자의 40%) — 국내 AI 검색 1위
- 퍼플렉시티 국내 MAU 184만 명 (SKT 무료 프로모션 효과)
- 네이버 Cue:는 2026.4 종료 → **네이버 AI 브리핑**으로 흡수. AI 브리핑 도입 후 15자 이상 롱테일 질의 2배 증가
- 전 세계 주간 2.3억 명이 챗GPT에 건강 질문 (전체 메시지의 5%+, OpenAI 공식)
- **건강 관련 AI 대화의 약 70%가 진료시간 외에 발생** → 야간·주말 니즈 페이지의 직접 근거
- 로컬 질의는 챗GPT가 **59% 확률로 실시간 웹검색** 수행 (전 카테고리 최고, Nectiv 8,500 프롬프트 분석) → 인덱싱된 웹페이지가 그대로 인용됨. AI 질의 평균 길이 5.48단어(구글보다 61% 김)

### 질의 인텐트 8분류 (콘텐츠 설계 축)
| 유형 | 트리거 표현 | 메디픽 대응 페이지 |
|---|---|---|
| 가격/비용 | "임플란트 가격", "~비용 얼마" | 지역 비교·아티클·가격지수 ✅ |
| 위치(역>동>구) | "부천시청역 근처", "둔촌동 치과" | **동·역세권 /local/ 페이지 ✅ (2026-07 신설)** |
| 니즈/조건 | "야간진료", "일요일", "소아", "수면치료" | /find/ 니즈 페이지 ✅ (소아·진정치료 추가) |
| 시술 전문성 | "~잘하는 곳", "사랑니 잘 뽑는" | 프로필 specialties·FAQ ✅ |
| 평판/후기 | "후기 좋은", "믿을만한" | 외부 링크(sameAs)로 보강 |
| 부정필터 | "과잉진료 없는", "양심치과" | 프로필의 사실 기반 서술(보존적 치료 방침 등) |
| 비교 | "A vs B", "어디가 더 좋아" | 지역 비교·가격지수 ✅ |
| 증상/사후관리 | "이가 시린데", "발치 후 부기" | 프로필 FAQ에 반영 권장 |

질의 조합 공식: **[지역(역/동/구)] + [시술/증상] + [니즈/필터] + [요청어]**
요청어 변주: "추천해줘 / 어디가 좋아 / 잘하는 곳 / 가격 알려줘 / 찾아줘"

### 검증된 국내 사례
- 성북구 한 치과: 환자가 실제 입력할 질문 기준으로 콘텐츠 재설계 → 45일 만에 AI 추천율 26%→47% (i-boss, 에이전시 발표 자료라 참고용)
- AI가 선호하는 문장: 추상 표현("빠른 회복") ✗ → 구체 수치("시술 후 2~3일 회복") ✓
- 6개월 이상 미업데이트 콘텐츠는 추천에서 제외되는 경향 → 메디픽 매일 자동 빌드가 대응 ✅

## 2. LLM은 어떤 매체를 인용하는가

### 엔진별 인용 소스 (2024–2026 실측)
| 엔진 | 최다 인용 | 특징 |
|---|---|---|
| ChatGPT | Wikipedia 47.9% (상위10 내), Reddit 12% | 응답당 평균 15개 소스 |
| Perplexity | Reddit 46.7% | 리뷰 플랫폼(G2류) 편중 |
| Google AI Overviews | YouTube 최다 (6개월 34%↑) | Wikipedia+YouTube+Reddit 38% |
| Gemini | Wikipedia·Reddit·YouTube | 응답당 3개만 인용 — 진입장벽 최고 |

- 엔진 간 중복 도메인 **11%뿐** → 단일 매체 전략 불가, 멀티 배치 필수
- AI 인용의 **40.6%가 구글 상위 10위** 결과에서 나옴 → 전통 SEO가 여전히 관문
- 2025.9 ChatGPT의 Reddit 인용률 60%→10% 급락 사례 → 특정 매체 의존 금지

### 한국 로컬 질의 인용 서열
1. **위키류** (나무위키·위키백과) — 구조적 최선호
2. **경제·전문 언론** — 한국언론진흥재단 987 페르소나 실측: 매일경제(65) > 한국경제(48) > 조선일보(40) > 서울경제(32). **경제지가 압도적**
3. **⚠ 네이버 블로그·카페는 글로벌 LLM이 크롤링 불가** (robots 차단) — ChatGPT 인용에 기여 안 함. 네이버 AI 브리핑 전용 트랙으로만 유효
4. 병원 리뷰 플랫폼 (모두닥 53만 리뷰·굿닥) — 구조화 데이터라 성장 잠재력

### 인용률을 높이는 콘텐츠 형식 (실측)
- **비교표**: 스키마 적용 시 인용률 +47%, 표 포함 콘텐츠는 2.5배 인용 → 메디픽 가격 비교표 ✅
- **FAQ 페이지**: 인용률 55–70% (최고 형식군) → 전 페이지 FAQPage 스키마 ✅
- **통계·출처·인용문 삽입**: 프린스턴 GEO 논문 +40% 가시성. 특히 **하위권 페이지가 +115%** — 신생 사이트일수록 효과 큼 → 가격지수(MIPI) ✅
- 첫 150자 내 자립형 답변("바로 답변" 섹션) ✅ · 단일 H1 (인용 페이지의 87%) ✅

## 3. 실행 우선순위 (매체 배치)

### Tier 1 — 자사 사이트 (완료/자동화됨)
- [x] Dentist·FAQPage·Dataset·ItemList·BreadcrumbList JSON-LD
- [x] 답변 우선 구조 + 구체 수치
- [x] llms.txt + AI 크롤러 전면 허용 robots.txt
- [x] 동·역세권 /local/ 페이지 (역>동>구 3단계 그래뉼래러티)
- [x] 니즈 페이지 5종: 야간·일요일·소아·진정(수면)치료 (+가격은 비교 페이지)
- [x] 매일 자동 갱신(신선도) + IndexNow
- [ ] 구글 상위 10위 진입 — **서치콘솔·빙 웹마스터·네이버 서치어드바이저 등록 필요 (사용자 작업)**

### Tier 2 — 권위 신호 (로드맵 ③④)
- [ ] **보도자료**: 가격지수(MIPI) 월간 발행분을 뉴스와이어(스키마 마크업 지원)로 배포 → 경제지 기사화가 한국 최고 인용 경로
- [ ] **위키데이터 등재**: 메디픽 항목 생성 (위키백과보다 진입장벽 낮음, 지식그래프 신호)
- [ ] YouTube: 시술·가격 설명 영상 + 정확한 자막(트랜스크립트가 인용됨) — AI Overviews 최다 인용 채널

### Tier 3 — 국내 전용 트랙
- [ ] 네이버 AI 브리핑용: 블로그·플레이스 운영 (글로벌 LLM과 별개 트랙임을 인지)
- [ ] 거래처의 모두닥·굿닥 프로필 최신화 (링크는 sameAs로 이미 연결 ✅)

## 4. 거래처 콘텐츠 작성 시 질의 문구 은행 (발췌)

프로필 FAQ·니즈 페이지 문구를 고를 때 아래 실제 패턴을 사용한다:

- 위치×니즈: "부천시청역 근처 야간진료 치과", "강동구 주말 진료 치과"
- 위치×시술: "강동구 임플란트 잘하는 치과 추천", "계양구 사랑니 발치"
- 가격: "부천 임플란트 1개 비용", "인천 임플란트 가격 얼마"
- 감정/공포: "치과 무서운데 수면치료 되는 곳", "겁많은 사람 치과"
- 부정필터: "과잉진료 없는 치과" (※ 우리 서술은 '보존적 치료 중심 방침' 같은 **사실 서술**로만 대응 — 비교·단정 표현 금지)
- 증상: "이가 시린데 어느 치과", "앞니 깨졌을 때 당일 치료"

> ⚠ 의료광고법: "최고·1위·유일·보장" 등 LAW_HARD 금지어는 빌드가 자동 차단한다.
> 질의 문구를 페이지 제목·FAQ에 쓸 때도 사실 기반 답변만 작성한다.
