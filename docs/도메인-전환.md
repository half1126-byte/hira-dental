# medipick.kr 도메인 전환 절차서

> 도메인 구매 완료 후 이 순서대로 진행. 사용자 작업 ①②⑤, Claude(자동화) 작업 ③④⑥.
> 전체 소요: DNS 전파 포함 반나절~1일.

## ① 도메인 구매 (사용자)

1. https://hosting.cafe24.com/ → 도메인 → `medipick.kr` 검색 → 구매 (연 1~2만원대)
2. 구매 시 네임서버는 카페24 기본값 그대로 두면 됨

## ② 카페24 DNS 레코드 설정 (사용자)

카페24 → 나의서비스관리 → 도메인 관리 → DNS 관리 → `medipick.kr`에 아래 5개 레코드 추가:

| 유형 | 호스트 | 값 |
|------|--------|-----|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| CNAME | www | half1126-byte.github.io |

## ③ 저장소 전환 (Claude — "도메인 샀어"라고 알려주면 실행)

1. `scripts/site-config.js`: `BASE_URL = 'https://medipick.kr'` (한 줄)
2. 루트에 `CNAME` 파일 생성 (내용: `medipick.kr`)
3. `.github/workflows/build.yml`: `_site` 복사 목록에 `CNAME` 추가
4. 정적 파일 URL 교체: `index.html`(canonical·og:url 2곳), `robots.txt`(Sitemap 1곳),
   `.github/workflows/partner.yml`(안내 링크 1곳), `docs/사용가이드.md`, `README.md`
5. 커밋 → 머지 → 빌드로 전체 페이지·sitemap·llms.txt 자동 재생성

## ④ GitHub Pages 설정 (Claude 안내 + 사용자 클릭)

1. https://github.com/half1126-byte/hira-dental/settings/pages
2. Custom domain에 `medipick.kr` 입력 → Save → DNS check 통과 대기(수 분~수 시간)
3. **Enforce HTTPS** 체크 (인증서 발급까지 최대 24시간)

## ⑤ 구주소 → 새주소 연결 확인 (자동)

- github.io 구주소는 GitHub이 자동으로 301 리다이렉트 → 기존 색인·링크 승계됨
- IndexNow가 다음 빌드에서 새 도메인 기준으로 전체 URL 재제출 (site-config에서 HOST 자동 파생)

## ⑥ 전환 후 재등록 (사용자, 한 번씩)

- Bing 웹마스터 / Google Search Console / 네이버 서치어드바이저에 `medipick.kr` 신규 등록 + sitemap 제출
- 보도자료 발송은 반드시 전환 완료 후 (원고 URL이 medipick.kr로 자동 갱신됨)
