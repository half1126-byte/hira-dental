# AILA 아이라 (AI가 찾아주는 우리 동네 병원) — 아주 쉬운 사용 가이드

계약한 치과를 등록하면 AI 검색(ChatGPT·Perplexity·Gemini 등)이 인용할 수 있는
프로필 페이지가 자동 생성되고 검색엔진에 통지됩니다. 여러분이 하는 일은 **입력과 버튼 클릭**뿐입니다.

## 0. 가장 쉬운 방법: 관리자 페이지 (추천)

https://half1126-byte.github.io/hira-dental/admin/ 에서 모든 작업을 폼으로 처리할 수 있습니다.

- 최초 1회: GitHub 토큰(Contents: Read and write) 발급해 붙여넣기 — 페이지 안에 안내 있음
- 등록: "＋ 새 거래처 등록" → 폼 입력 → 저장 (금지어 자동 검사 포함)
- 수정: 목록에서 "수정" → 가격표·FAQ를 표 형태로 입력
- 해지/재게시: 목록에서 버튼 한 번
- 저장하면 자동 빌드·배포·검색엔진 통지까지 진행, 상단에 배포 상태 표시

아래 1~3번은 GitHub 화면으로 하는 예비 방법입니다.

## 1. 거래처 등록 (계약했을 때) — 입력 1분 + 자동 10분

1. ① 관리 버튼 페이지 → **Run workflow** 클릭
2. `action`은 `add` 그대로, `name`에 치과 이름, `sido`에서 지역 선택 (나머지 칸은 비워도 됨)
3. 초록 **Run workflow** 클릭 → 끝
4. 약 10분 뒤 홈페이지 `/clinics/`에 프로필 생성. HIRA 공공데이터 조사,
   AI 인용용 구조화 페이지 생성, Bing·네이버 통지까지 전부 자동.

> ⚠️ 페이지에 "게재에 경제적 대가가 포함될 수 있습니다" 고지가 자동으로 붙습니다.
> **계약(동의)이 확정된 치과만** 등록하세요.

## 2. 가격·FAQ 채우기 — 인용 확률을 올리는 핵심

AI는 구체적 숫자가 있는 페이지를 훨씬 잘 인용합니다. 계약 시 가격표를 받아 넣으세요.

1. ③ 거래처 정보 수정 → 연필 아이콘(Edit)
2. 해당 치과의 `"prices"` 대괄호 안에 추가:
   `{ "item": "임플란트(국산)", "amt": 1200000, "note": "뼈이식 별도" }`
3. `"hours"`(진료시간), `"faq"`(질문·답변)도 같은 방식 — 맨 위 "예시치과의원" 항목이 견본
4. **Commit changes** 클릭 → 다음 자동 빌드(매일 새벽 1시) 때 반영
   (즉시 반영: ① 관리 버튼에서 `activate` + 해당 id 실행)

> ⚠️ "최고", "1위", "유일", "완치", "100%" 같은 과장 표현을 쓰면
> 의료광고법 보호 장치가 **빌드를 자동 중단**시킵니다. 사실만 적으세요.

## 3. 거래처 해지 (계약 끝났을 때)

1. ① 관리 버튼 → **Run workflow**
2. `action`을 `pause`로, `id`에 프로필 주소 끝부분 입력 (예: `gangdong-trium-dental`)
3. 실행 → 해지일 기록 + 페이지 자동 제거. 재계약 시 `activate`로 즉시 복구.

## 4. 성과 확인

- **매주 월요일 오전 10시** 인용 점검 리포트가 `reports/` 폴더에 자동 저장
- 직접 확인: ChatGPT·Perplexity에 "둔촌동 치과 추천해줘" → 출처에 우리 사이트 있는지 확인
- 색인까지 **1~2주** 걸림 — 등록 직후 안 나오는 건 정상

## 자동으로 되는 일 (건드릴 필요 없음)

- 매일 새벽 1시: HIRA 최신 데이터로 전체 사이트 재생성·재배포
- 매 배포 후: Bing·네이버 자동 통지 (IndexNow)
- 등록 시: HIRA에서 주소·전화·좌표 자동 조사
- 항상: 금지어 검사, 고지 문구, AI용 구조화 데이터 자동 처리
- HIRA 신고 기관이면 65개 지역 아티클에서 자동 링크

## 문제가 생겼을 때

| 증상 | 해결 |
|------|------|
| 실행이 빨간 X (실패) | X 클릭해 원인 확인. 대부분 ① 금지어 → 문구 수정 후 재실행 ② 중복 등록 → `activate` 사용 |
| 홈페이지에 안 보임 | 10분 기다린 뒤 Ctrl+F5. 그래도 없으면 실행 성공(초록) 여부 확인 |
| AI 답변에 안 나옴 | 색인 1~2주 대기 → 가격·FAQ 보강 → 치과 홈페이지·네이버 플레이스에서 프로필로 링크 요청 |
