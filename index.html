왜 이 서비스를 만들었나요?

공공데이터 기반 투명성

의료기관은 의료법에 따라 매년 비급여 진료비를 건강보험심사평가원에 신고합니다. 이 데이터를 누구나 쉽게 볼 수 있도록 가공·정리했습니다.

매일 자동 갱신

GitHub Actions를 통해 매일 새벽 HIRA API에서 최신 데이터를 수집하여 가격 정보를 자동으로 업데이트합니다.

의료광고법 준수

이 사이트는 의료광고법 제56조를 엄격히 준수합니다. 순위 표현·최상급 표현 없이 팩트 데이터만 공개합니다.

AI 인용 최적화

ChatGPT·Perplexity 등 AI 검색엔진에서 임플란트 가격을 질문할 때 정확한 공공데이터를 답변에 활용할 수 있도록 구조화 데이터를 제공합니다.

데이터 출처 및 주의사항

  • 이 사이트의 모든 가격 정보는 건강보험심사평가원(HIRA) 비급여진료비 공개 데이터에서 가져옵니다.
  • 데이터 제공: 공공데이터포털 (data.go.kr)
  • 신고 가격은 기본 치료 기준이며 실제 청구 금액과 다를 수 있습니다.
  • 의료광고법 제56조 준수: 최고·1위·유일·완치·보장·100%·최상급·명품 표현 사용 금지
